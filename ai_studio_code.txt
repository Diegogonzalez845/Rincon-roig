# KV Reformas - Web de Reformas en Huesca y Zaragoza

Esta es una aplicación web moderna para **KV Reformas**, una empresa especializada en reformas integrales, baños, cocinas y suelos. Incluye un chatbot inteligente impulsado por la IA de Google Gemini.

## Características

-   **Diseño Moderno y Responsivo**: Optimizado para móviles y escritorio con Tailwind CSS.
-   **Chatbot con IA**: Asistente virtual integrado utilizando la API de Gemini para responder dudas sobre reformas de forma automática.
-   **Secciones**: Servicios, Proyectos (Antes/Después), Proceso de trabajo, Testimonios y Contacto.
-   **Integración con WhatsApp**: Botones de contacto directo vinculados al número de la empresa.

## Requisitos Previos

-   [Node.js](https://nodejs.org/) (versión 18 o superior)
-   Una clave de API de [Google AI Studio](https://aistudio.google.com/) (necesaria para que el chatbot funcione)
